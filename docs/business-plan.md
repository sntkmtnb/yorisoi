# よりそい - 事業計画書

## エグゼクティブサマリー
「よりそい」は、40代以上に特化したAIコンシェルジュ型出会いサービス。
AIが会話を通じてユーザーの人柄を理解し、毎日一人の最適な相手を紹介する。
従来のスワイプ型マッチングアプリとは根本的に異なる「ゼロ負担」のUXで、
40代以上の未開拓市場を獲得する。

## 市場機会
- 日本のマッチングアプリ市場: 1,094億円（2026年）、年7%成長
- 40代以上の独身者: 約1,500万人
- 40代以上に特化したAI活用サービス: **ゼロ**（市場空白）
- 40代の可処分所得は20-30代の1.5-2倍 → 高単価が成立

## プロダクト

### コア機能
1. **AIコンシェルジュ** — 会話で人柄を理解、プロフィール自動生成
2. **デイリーマッチ** — AIが選んだ1日1人の紹介
3. **手紙機能** — マッチ後の丁寧なメッセージ交換
4. **相談機能** — デート相談、恋愛アドバイスをAIが提供

### 技術
- Next.js (PWA) + Supabase + OpenAI GPT-4o
- サーバーレス構成で低コスト運用
- AI推論コスト: ~¥5/会話（GPT-4o miniで更に削減可能）
