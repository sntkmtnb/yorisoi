# よりそい - 競合分析レポート

## 日本のマッチングアプリ市場

### 市場規模
- **2025年**: 1,023億円
- **2026年予測**: 1,094億円（前年比7%増）
- **2028年予測**: 約2,700億円（2021年比1.8倍）
- 新規参入企業: 2019年の5社 → 2025年は28社（6年で5.6倍）

### 40代以上の市場機会
- マッチングアプリ市場の成長は20-30代が牽引
- **40代以上は未開拓の巨大セグメント**
- 40代以上の独身者は約1,500万人（推定）
- 可処分所得が高い → 高単価サービスが成立する

---

## 主要競合

### 1. マリッシュ (marrish) — 最大の直接競合
- **累計会員数**: 400万人
- **年齢層**: 40代が44.6%（最多）
- **特徴**: 再婚・バツイチに特化
- **料金**: 男性 ¥3,400/月〜、女性無料
- **強み**: 再婚理解層に強い支持、40-50代が多い
- **弱み**: 
  - UIが従来型（スワイプ/いいね）
  - 再婚特化で初婚40代は入りにくい
  - AI活用なし
  - プロフィール作成の負担は従来通り

### 2. Pairs (ペアーズ) — 最大手
- **累計会員数**: 2,000万人以上
- **年齢層**: 20-30代が中心（40代は約15%）
- **料金**: 男性 ¥3,700/月〜、女性無料
- **弱み**: 40代はマイノリティで不利、若者向けUI

### 3. Omiai
- **年齢層**: 30代が44%、20-30代で87%
- **特徴**: 婚活寄り
- **弱み**: 40代以上はほぼ対象外

### 4. youbride (ユーブライド)
- **年齢層**: 30-40代中心
- **特徴**: 真剣婚活
- **弱み**: 古いUI、AI活用なし

### 5. アンジュ
- **年齢層**: 30代以上限定
- **特徴**: 大人向け
- **弱み**: 知名度低い、機能が限定的

## よりそいの競争優位

### 1. AIネイティブ（唯一の差別化）
- 2026年時点で日本の出会い系にAIコンシェルジュを本格導入しているサービスは**ゼロ**
- 先行者利益が取れる
- 技術的な参入障壁になる

### 2. 完全40代以上特化
- マリッシュは「再婚」特化だが、40代全体をカバーしていない
- 初婚40代、恋活40代、友活40代のニーズが未充足

### 3. ゼロ負担UX
- 既存アプリは全て「ユーザーが頑張る」設計
- よりそいは「AIが全部やる」設計
- 40代の「めんどくさい」を完全に解消

### 4. 男女同額モデル
- 既存アプリは「男性が払う」モデル → 不平等感
- 同額 → 対等な関係性、質の高いユーザー層

## リスクと対策

### リスク1: 大手が追随
- Pairsなどが40代向けAI機能を追加する可能性
- **対策**: 先行して40代コミュニティを構築。ブランドと信頼を確立

### リスク2: ユーザー獲得コスト
- 出会い系のCPA（顧客獲得単価）は高い（¥3,000-10,000）
- **対策**: コンテンツマーケ（SEO）で自然流入。口コミ設計

### リスク3: 法規制
- インターネット異性紹介事業の届出が必要
- **対策**: 早期に届出。年齢確認・本人確認を厳格に

### リスク4: AI精度
- マッチング精度が低いと離脱
- **対策**: ユーザーフィードバックを学習に活用。初期はAI + 手動キュレーション

**40代以上 × AIコンシェルジュ** のポジションは完全に空白。
市場は成長中で、競合は若者向けに集中している。
